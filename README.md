# anfisa.link

Портфолио Анфисы Уткиной — статический сайт.

Опубликован на [anfisa.link](https://anfisa.link) через GitHub Pages.

## Локальный запуск

```bash
python3 -m http.server 8765
```

Открыть [http://localhost:8765/](http://localhost:8765/).

## Стек

- Чистый HTML + инлайн-CSS, без сборки и зависимостей.
- Шрифт Inter с Google Fonts.
